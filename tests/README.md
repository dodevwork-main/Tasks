# Tasks - Tests

<a href="https://taskstests.vercel.app/">Example with Vercel</a>

- Create React App
- Jest
- React-Testing-Library
- Cypress
- ESLint
- Prettier

<br>

Run `test`

```shell
npm run test
```

Run `test` with `coverage`

```shell
npm run test:coverage
```

---

Run e2e test `cypress`

```shell
npm run start
npm run cypress
```

---

Run `eslint`

```shell
npm run lint
```

Run `eslint` with `fix`

```shell
npm run lint:fix
```

---

Run `prettier` with `write`

```shell
npm run prettier
```

Run `prettier` with `check`

```shell
npm run prettier:check
```

---

Run for `dev`

```shell
npm install
npm run start
```

Run for `prod`

```shell
npm install
npm run build
```
